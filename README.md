
# Wazuh SOC Lab

## Overview

This project demonstrates the implementation of a Security Operations Center (SOC) lab using Wazuh, Sysmon, Kali Linux, and Windows 10. The objective is to simulate real-world cyber attacks, detect them using Wazuh, and investigate the generated alerts from the perspective of a SOC Analyst.

Rather than focusing only on offensive techniques, this project emphasizes the complete incident detection and investigation lifecycle.

---

## Objectives

- Build a fully functional SOC lab.
- Deploy Wazuh as the Security Information and Event Management (SIEM) platform.
- Configure Sysmon to collect detailed Windows telemetry.
- Simulate multiple cyber attacks.
- Analyze alerts generated by Wazuh.
- Investigate security events using SOC analyst methodologies.
- Build custom Wazuh rules.
- Map attacks to the MITRE ATT&CK Framework.
- Recommend appropriate mitigation strategies.
